googleface.py程式說明:
valid 資料庫圖片
compares 要與valid資料庫比對的圖片
validPicPath 放圖片的資料夾位置
model_path 模組路徑
路徑不能有中文

會比對valid跟每個compares，並顯示特徵值差距
